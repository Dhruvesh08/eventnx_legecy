Setting Up Login Button:

Place below script in head section of your webpage

<script src='https://www.eventnx.com/eventscript/eventproloader.js'></script>

Place below code on webpage where you want to show LinkedIn login button

<div class='linkedinlogin' id='linkedinlogin_be7b8da0-de11-4d48-8c35-25b7f642b5ac'></div><div id='divallowpost'><input type='checkbox' id='AllowPost' checked='checked' /> Let my social network know that I am attending the event. This gives us permission to post to your profile about your attendance. It also allows other registered visitors to see that you’re registered to attend.</div>

Setting Up Event Registration Form:

Place below code in header section only if you do not have jquery installed on your webpage

<script src='https://www.eventnx.com/eventscript/jquery-1.12.0.min.js'></script>

Place below script in header section of your webpage

<script src='https://www.eventnx.com/eventscript/eventproloader.js'></script>

Use below hidden fields value to autofill form with LinkedIn data

  • hidEventProUserId
  • hidProfileImage
  • hidFirstName
  • hidLastName
  • hidEmail

Place below div tag in Registration Form

<div id='eventregistration' class='eventregistration' > Place above hidden fields value to display /div>

Setting Up Confirmation Page:

Place below code in header section only if you do not have jquery installed on your webpage

<script src='https://www.eventnx.com/eventscript/jquery-1.12.0.min.js'></script>

Place below script in header section of your webpage

<script src='https://www.eventnx.com/eventscript/eventproloader.js'></script>

After successful registration, render below script with CRM parameters.

<script src= 'https://www.eventnx.com/eventscript/submitdatatoeventpro.js' data-eventprouserid='Eventpro_system_userid' data-firstname='Your_system_firstname_param' data-lastname='Your_system_lastname_param' data-emailid='Your_system_emailid_param' data-companyname='Your_system_companyname_param' data-jobtitle='Your_system_jobtitle_param' data-crmregid='Your_system_crmregid_param'>

Below script will display a sharable link on the confirmation page which registered user can copy and share on social media.

<div id='divsuccessmessage' style='display: none;'></div>
